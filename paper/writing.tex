\section{Notation}
\begin{enumerate}
    \item  Use bold letters for vectors and regular letters for
    scalars. For instance, let $n$ denote the sample size and $\bv$ denote the vector that collects the parameters of interest.
    
    \item Use bold capital letters to denote matrices. For instance, let $\bW \in \RR ^ {d \times K}$ denote the weight matrix. 
\end{enumerate}
\section{Mathematical display}
    \begin{enumerate}
    \item Use \verb|^{\top}| to denote the transpose of a vector or matrix. For instance,
    the transpose of matrix $\bA$ is $\bA^{\top}$.
    \item Use \verb|\balpha^{\top}\bbeta| to  denote the inner product between two vectors $\balpha$ and $\bbeta$. For instance,
    let $\balpha$ and $\bbeta$ be two vectors of dimension $d \times 1$, then its inner product is given by $\balpha^{\top}\bbeta$. 
    \item Use \verb|$\texttt{}$| to refer to a package or a specific function therein. For instance,
       the $\texttt{Pytorch}$ library adds the bias term by default, or
    be sure to do $\texttt{optimizer.step()}$ before $\texttt{scheduler.step()}$. 
    
    \item Don't use \verb|$\texttt{}$| when referring to an approach or algorithm; for example, Principal Component Analysis (PCA).
    \item Use \verb|$\mathrm{}$| to refer to math functions. For instance,
     the $\mathrm{LogSoftmax()}$ function returns the logarithm of the $\mathrm{Softmax()}$ function.
    \item Do not use \verb|$\cdot$| for any kind of multiplication, e.g., scalar-scalar product, Hadamard product, inner product, matrix-vector product, etc. 
    \item Use \verb|$\log_{base}(number)$| instead of \verb|$log_{base}(number)$| to denote logarithm. For instance,
    the objective function we want to minimize is $\mathrm{Loss}:= -\log(q_j(x))$
         ,where $q_j(x)$ is obtained by applying the $\mathrm{Softmax()}$ function to the raw scores.
    
\end{enumerate}
\section{Word choice}
\begin{enumerate}
    \item There is no multi-dimensional vector; instead we should use the term multi-dimensional tensor.
    \item Instead of using ``number of samples'', we should use ``number of observations'' or ``sample size'' to describe how many observations we have.
    \item Never use ``want to''. Use ``plan to'', ``wish to'', ``intend to'', etc. 
\end{enumerate}
\section{Organization}
\begin{enumerate}
    \item Define a quantity before using it.
     \item Explain the domain and image space first when describing a function. For example, consider $f: \RR \to \RR ^ d$. 
\end{enumerate}
